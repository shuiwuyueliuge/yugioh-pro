  • [废品收集者]
    <ジャンク·コレクター>
    [10/04/13]

    ●把场上表侧表示存在的这张卡和自己墓地存在的1张通常陷阱卡从游戏中除外发动。这张卡的效果变成和为这个效果发动而从游戏中除外的通常陷阱卡的效果相同。这个效果在对方回合也能发动。
    ◇诱发即时效果(进入连锁)。
    ◇任意发动。
    ◇把场上表侧表示存在的这张卡和自己墓地存在的通常陷阱卡从游戏中除外是COST。
    ◇选择除外的陷阱卡是取对象效果的场合，在这个效果发动时选择对象。
    ◇伤害步骤不可以发动。
    ◇满足那张陷阱卡发动条件的场合才能发动。
    ◇不需要支付那张陷阱卡发动的COST。
    ◇可以在这张卡召唤、特殊召唤成功时，发动这个效果来使用「激流葬/激流葬」等的效果。
    ◇这张卡获得的陷阱卡效果视为效果怪兽的效果。
    例1
    「仪式之槛/仪式の槛」在场上存在并效果适用中
    「废品收集者/ジャンク·コレクター」将「神圣防护罩-反射镜力-/圣なるバリア－ミラーフォース－」除外并获得其效果
    场上的「纳祭之魔/サクリファイス」不会被破坏。
    例2
    「废品收集者/ジャンク·コレクター」将「激流葬/激流葬」除外并获得其效果
    「元素英雄 荒野侠/E·HERO ワイルドマン」会被破坏
    ◇不可以获得「附锁链的爆弹/锁付き爆弾」等使用效果后会留在场上的通常陷阱卡的效果。
    ◇不能获得「伪陷阱/伪物のわな」的效果。
    ★能否获得「伪羽/フェイク·フェザー」的效果，调整中。
    ★在对方怪兽攻击宣言时发动这张卡效果获得「废铁稻草人/くず鉄のかかし」的效果让对方攻击无效后，获得的『发动后这张卡不送墓地，直接盖伏』这个效果如何处理？调整中
    ◇自己场上只有包括这张卡以内的2只怪兽表侧攻击表示存在时，对方发动「神圣防护罩-反射镜力-/圣なるバリア－ミラーフォース－」的场合，自己不能将这张卡以及自己墓地存在的1张「星光大道/スターライト·ロード」从游戏中除外发动这个效果
    ★这张卡获得的持有「把场上的牌破坏的效果」的效果发动的场合，能否对应那个效果连锁发动「星尘龙/スターダスト·ドラゴン」的效果？调整中
    ◇自己墓地只存在「狡猾的落穴/狡猾な落とし穴」的场合，不能将其除外。
